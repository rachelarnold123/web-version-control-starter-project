HTML is how programmers write websites. HTML stands for Hyper Text Markup Language. It is the standard language for creating webpages in markup. It helps describe the structure of a webpage. HTML consists of a series of elements that tells the brower to to display it. As well as automatically labeling context like "this is a paragraph"

Bootstrap allows you to write easier HTML Files. When you use Bootstrap in HTMl you are using prewritten CSS to quickly and efficivly add style or functionality to your website
