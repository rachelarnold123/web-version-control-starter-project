HTML Made Easy

by Rachel Arnold

This webpage is a intro to HTML & BootStrap in HTML. By the end of this you will be able to create a website just like this one! If you are new to HTML or Bootstrap, read the drop down menus below!

HTML is how programmers write websites. HTML stands for Hyper Text Markup Language. It is the standard language for creating webpages in markup. It helps describe the structure of a webpage. HTML consists of a series of elements that tells the brower to to display it. As well as automatically labeling context like "this is a paragraph"

Bootstrap allows you to write easier HTML Files. When you use Bootstrap in HTMl you are using prewritten CSS to quickly and efficivly add style or functionality to your website

Headings

In order to title your website, divide your website in sections, and make it easy to read for the eyes you need to add headings. Luckly for all of us, they are pretty easy to make once you understand how! At the beginning of the heading add <h1> and at the end of the heading add <h1/>
When put into use should look like the following
<h1>This is how you write a Heading in HTML =D<h1/>
Its also important to note you can use other numbers besides 1 to make a heading smaller. Heading 1 is the biggest, while Heading 2 is smaller, etc

Images Made Easy

Websites are not complete without images so how do you add them? To add your first image, follow the step by step guide below!

  1. Find a free(not copyrighted) jpeg image online and download it(You can also name it something easy to remeber like "computer.jpg)
  2. Create a folder in your website's folder for images
  3. Move your new image into the image folder of your website
  4. Go into your index.html folder
  5. Add the image onto your website by doing the following

    6. Start by writing "img", for src= put the directory to the image, and for alt= write a breif description of what the image is showing. Make sure to put img src="Images/computer.jpg" alt="Picture of working website on computer" width="304" height="236"

  6. Finally once you load your website you should have a image! If this doesnt work, please refer to "HTML Image Help" in Resources
